####################
# Naming guides:
# - Always try to find attested historical names
# - When no historical name is available, you can translate the original name or use an older version, but be *very* careful
# - Try to include the source where you found the name, if you think it is useful, in the README.md or in a forum comment
####################
# Formatting rules:
# 1.) Always save this using the WINDOWS-1252 character encoding
# 2.) Use proper indentation for each level. (4 spaces, no tabs)
# 3.) All names should be put inside of quotes (" ")
# 3.) This file should contain only the titles that this mod localises
# 4.) The titles should be in the same order as they are in the original SWMH file
# 5.) You should always write comments for each name when:
#       - The name contains characters that the game does not support (Not in the WINDOWS-1252 charset)
#       - The name has other attested alternatives
#       - The name is not historical or it is subject of controversy
####################
# Notes:
# - For all romanian names: Every single "sh" is actually a s with , under it.
# - Nahua (aztec) names are not historical (obviously) but encouraged.
